# Project Roadmap

This repository will develop the ITI–gITI–IDI framework through a staged applied case study of Lehman Brothers.

## Phase 1 — Public Source Collection

Collect public and verifiable materials, including financial statements, bankruptcy examiner materials, regulatory records, market data, and timeline evidence.

Output:

- source list;
- document table;
- preliminary evidence map.

## Phase 2 — Variable Mapping

Map collected evidence into the three framework layers:

- ITI: expected institutional strain;
- gITI: realized geometric tension;
- IDI: distortion gap between the two layers.

Output:

- variable table;
- source-to-variable map;
- preliminary coding notes.

## Phase 3 — Index Construction

Convert mapped variables into comparable scores.

Output:

- normalization method;
- preliminary ITI score;
- preliminary gITI score;
- preliminary IDI result.

## Phase 4 — Lehman Brothers Case Report

Prepare a short applied case report showing how institutional distortion appeared across formal expectation, market reaction, and systemic outcome.

Output:

- case timeline;
- index table;
- interpretation note;
- visual summary.

## Phase 5 — Extension

After the Lehman Brothers case, the same model may be applied to other institutional, financial, regulatory, or public administration cases.
